# eto -- bleh!

A small updater.

![bleh](bleh.png)

## Usage

### eto-package

```
eto-packager --old ./my-old-dir --new ./my-new-dir -p ./package.zip
```

### eto-updater

Place in target directory along with the package zip and run.
Updater automatically finds the package and applies.
